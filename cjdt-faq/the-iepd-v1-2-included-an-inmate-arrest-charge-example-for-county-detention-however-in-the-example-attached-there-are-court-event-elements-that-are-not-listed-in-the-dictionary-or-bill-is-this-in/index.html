The IEPD v1.2 included an inmate/arrest/charge example for County Detention. However, in the example (attached) there are court event elements that are not listed in the dictionary or bill. Is this information now required?

Created:June 11, 2020
Updated:July 31, 2020

  • The IEPD v1.2 included an inmate/arrest/charge example for County Detention. However, in the example (attached) there are court event elements that are not listed in the dictionary or bill. Is this information now required?
< Back

Court event elements are not required for County Detention (CJDT CD). The FDLE_Mapping_Spreadsheet (Data tab, row 25) indicates the court event elements are associated with arrest data for CJDT CD. This is an error. The court event elements are correctly associated with CJDT CoC (rows 124, 250-254). Moving forward we will remove the CJDT CD reference from the mapping spreadsheet and correct the example in the zipped file.
